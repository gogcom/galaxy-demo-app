This is a demonstration project for Galaxy SDK.

Supported Galaxy SDK: 1.138.1

To compile this project you need the CMake utility.

Step by step configuration:
1. Download an appropriate development kit package from https://devportal.gog.com (you should register as a game developer at GOG first)
2. Unpack contents of the development kit package
3. Copy Local.cmake.template file and save it as Local.cmake
4. Open Local.cmake and set path to development kit package
3. Configure and generate CMake build
4. You might want to change CLIENT_ID and CLIENT_SECRET in GalaxyDemo.cc to your own (you need to get it from GOG)
5. Compile
6. Run the GalaxyDemo executable
